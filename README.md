## 项目简介

这是 Renomee AI https://xiaojingjia.top 智能文件重命名工具的官方网站源码，基于 React + TypeScript + Tailwind CSS 开发。

**Renomee** 是一款革命性的文件重命名工具，通过 AI 技术让文件管理变得简单高效。

## 🚀 核心特色

- 🤖 **AI 智能识别** - 自动识别文件内容，生成有意义的文件名
- 🗣️ **自然语言操作** - 用说话的方式描述规则，无需学习正则表达式
- ⚡ **批量高效处理** - 一次处理数百个文件，效率提升 10 倍
- 🔒 **数据安全可靠** - 本地处理，文件不上传，隐私有保障
- 📱 **跨平台支持** - 支持 Windows 10+ 和 macOS 10.15+

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **样式方案**: Tailwind CSS 3.x
- **路由管理**: React Router DOM 6
- **构建工具**: Create React App
- **部署方案**: Docker + Nginx
