# 快速开始

本指南将帮助您在 5 分钟内开始使用 Renomee AI。

## 📥 下载与安装

### Windows 用户

1. 访问 [官方网站](https://renomeeai.com/) 
2. 点击 **"下载 Windows 版本"** 按钮
3. 下载完成后，双击 `Renomee-Setup.exe`
4. 按照安装向导提示完成安装

!!! tip "系统要求"
    - Windows 10 或更高版本
    - 64位操作系统
    - 至少 100MB 可用空间

### macOS 用户

1. 访问 [官方网站](https://renomeeai.com/)
2. 点击 **"下载 macOS 版本"** 按钮
3. 打开下载的 `.dmg` 文件
4. 将 Renomee 拖动到应用程序文件夹

!!! tip "系统要求"
    - macOS 10.15 (Catalina) 或更高版本
    - Apple Silicon 和 Intel 芯片均支持
    - 至少 100MB 可用空间

## 🎯 第一次使用

### 步骤 1：启动应用

安装完成后，从开始菜单（Windows）或启动台（macOS）启动 Renomee。

### 步骤 2：选择文件

有三种方式添加文件：

=== "方式 1：拖放"
    直接将文件或文件夹拖放到 Renomee 窗口中。

=== "方式 2：选择文件夹"
    点击 **"选择文件夹"** 按钮，浏览并选择要处理的文件夹。

=== "方式 3：选择文件"
    点击 **"选择文件"** 按钮，选择一个或多个文件。

!!! example "示例"
    假设您有一个包含照片的文件夹：
    ```
    📁 旅行照片/
       ├── IMG_0001.jpg
       ├── IMG_0002.jpg
       └── IMG_0003.jpg
    ```

### 步骤 3：输入重命名规则

在规则输入框中，用自然语言描述您的需求：

!!! success "推荐规则示例"
    === "按日期命名"
        ```
        按拍摄日期命名，格式：2025-01-26
        ```
        
        **结果：**
        ```
        IMG_0001.jpg → 2025-01-26_001.jpg
        IMG_0002.jpg → 2025-01-26_002.jpg
        ```

    === "添加前缀"
        ```
        给所有文件添加前缀"旅行_"
        ```
        
        **结果：**
        ```
        IMG_0001.jpg → 旅行_IMG_0001.jpg
        IMG_0002.jpg → 旅行_IMG_0002.jpg
        ```

    === "替换文本"
        ```
        把 IMG 替换成 Photo
        ```
        
        **结果：**
        ```
        IMG_0001.jpg → Photo_0001.jpg
        IMG_0002.jpg → Photo_0002.jpg
        ```

### 步骤 4：预览结果

点击 **"预览"** 按钮，查看重命名后的效果：

```
✓ 将要重命名 3 个文件

原文件名              →  新文件名
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
IMG_0001.jpg         →  2025-01-26_001.jpg
IMG_0002.jpg         →  2025-01-26_002.jpg
IMG_0003.jpg         →  2025-01-26_003.jpg
```

!!! warning "重要提示"
    预览模式不会修改文件，可以放心查看。如果结果不满意，可以修改规则重新预览。

### 步骤 5：执行重命名

确认无误后，点击 **"执行重命名"** 按钮。

✅ **完成！** 您的文件已成功重命名。

## 🎓 进阶技巧

### 使用变量

Renomee 支持强大的变量系统：

| 变量 | 说明 | 示例 |
|------|------|------|
| `{date}` | 文件修改日期 | 2025-01-26 |
| `{time}` | 文件修改时间 | 14-30-00 |
| `{name}` | 原文件名（不含扩展名） | IMG_0001 |
| `{ext}` | 文件扩展名 | jpg |
| `{n}` | 序号（从1开始） | 001, 002, 003 |
| `{parent}` | 父文件夹名 | 旅行照片 |

!!! example "组合使用变量"
    规则：`{parent}_{date}_{n}{ext}`
    
    **结果：**
    ```
    IMG_0001.jpg → 旅行照片_2025-01-26_001.jpg
    IMG_0002.jpg → 旅行照片_2025-01-26_002.jpg
    ```

### AI 智能模式

启用 AI 模式后，Renomee 会分析文件内容：

```
照片文件 → 自动识别拍摄地点、主题
文档文件 → 提取标题和关键词
音视频 → 识别标题、艺术家信息
```

查看 [基础使用指南](guide/basic-usage.md) 了解更多 AI 功能。
