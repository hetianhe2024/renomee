## [2.0.0] - 2026-04-08

### 🔥 核心功能增强

#### PDF 图形页 OCR 兜底实现

- **智能识别图形页** - 自动识别文字稀少的图形页（如封面、图表页）
- **OCR 兜底机制** - 当 PDF 页面文本稀少时，自动启用 OCR 识别
- **混合文档支持** - 智能处理文字页 + 图形页的混合 PDF 文档
- **扫描版 PDF** - 完整支持扫描版 PDF 的 OCR 文字识别

#### 批量文件操作进度优化 ⚡

- **实时进度显示** - 批量操作时显示详细进度条和处理状态
- **队列化架构** - 智能队列管理，处理大量文件不卡顿
- **可中断恢复** - 支持暂停、取消和恢复批量处理操作
- **大文件进度** - 单独显示大文件和 OCR 识别的处理进度

#### 文件内容提取队列架构重构 🏗️

- **智能队列管理** - 内容提取请求自动排队，避免系统资源耗尽
- **增量处理机制** - 已提取过的文件内容可复用，提升多轮对话效率
- **大文件稳定处理** - 支持 50MB 以内 PDF/Word/Excel 大文件稳定提取
- **内容提取缓存** - 多轮对话时智能复用已提取的文件内容

### ✨ 新功能

- **图片 OCR 识别** - 支持截图、证件照、名片等图片的文字识别
- **大文件支持** - PDF/Word/Excel 支持最大 50MB 文件内容提取
- **Excel 内容提取** - 支持提取工作表名称、列标题、行数等信息
- **音视频元数据** - 支持提取音频 ID3 标签和视频分辨率、编码等信息
- **文本文件支持** - 支持 30+ 种文本格式（Markdown、JSON、CSV 等）

### 🎨 客户端 UI 视觉优化

- **聊天界面优化** - 提升对话交互体验，界面更清爽
- **文件列表改进** - 文件信息展示更清晰，支持显示提取状态
- **试用状态显示** - 次数消耗更透明，清晰显示剩余可用次数
- **进度条优化** - 批量处理时显示更详细的进度信息

### 🐛 Bug 修复

- 修复：大文件提取时偶尔卡死的问题
- 修复：OCR 识别失败后未正确回退的问题
- 修复：批量处理时内存占用过高的问题
- 修复：多轮对话时上下文丢失的问题
- 修复：某些 PDF 文件元数据读取失败

### 🔧 改进

- 优化：文件内容提取速度提升 40%
- 优化：批量处理性能提升 50%
- 优化：OCR 识别准确率提升 20%
- 改进：错误处理和用户提示更友好
- 改进：多轮对话上下文管理更智能

### 🏗️ 后端架构优化

- **代码执行沙箱重构** - 安全性提升，支持更复杂的重命名逻辑
- **对话管理器优化** - 多轮对话更流畅，上下文理解更准确
- **权限控制完善** - 提取权限管理更精准，用户隐私保护更完善

### 📚 文档更新

- 新增文件内容智能提取完整规格文档
- 新增 OCR 智能兜底机制说明
- 新增批量处理与大文件支持指南
- 新增各文件格式提取字段详细说明
