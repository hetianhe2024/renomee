# 更新日志

本页记录 Renomee 的所有版本更新和变化。

## 版本号说明

版本号格式：`主版本号.次版本号.修订号`

- **主版本号：** 重大功能更新或架构变化
- **次版本号：** 新功能添加
- **修订号：** Bug 修复和小改进

---

## [1.2.0] - 2025-01-26

### ✨ 新功能

- **AI 智能模式增强**
  - 新增图片场景识别（风景、人像、动物等）
  - 支持文档关键词自动提取
  - 优化 AI 识别速度（提升 30%）

- **批量处理优化**
  - 支持 CSV 批量导入重命名规则
  - 新增分组处理模式
  - 支持递归处理子文件夹

- **变量系统扩展**
  - 新增 `{exif:GPS}` GPS 位置变量
  - 新增 `{doc:Pages}` 文档页数变量
  - 支持自定义变量格式

### 🐛 Bug 修复

- 修复：特殊字符文件名导致程序崩溃
- 修复：网络驱动器上文件重命名失败
- 修复：macOS 上撤销功能偶尔失效
- 修复：大文件（>2GB）处理内存泄漏

### 🔧 改进

- 优化：启动速度提升 40%
- 优化：内存占用降低 25%
- 改进：预览界面 UI 更新
- 改进：错误提示更友好
