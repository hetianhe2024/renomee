# 隐私政策

**生效日期：** 2024年10月15日  
**最后更新：** 2025年01月26日

感谢您使用 Renomee AI 智能文件重命名工具。我们非常重视您的隐私和数据安全。

## 📋 政策概要

**Renomee 的核心原则：**

- ✅ **本地优先** - 所有文件在您的设备上处理
- ✅ **最小收集** - 只收集必要的功能数据
- ✅ **透明公开** - 清楚告知收集的数据和用途
- ✅ **用户控制** - 您可以随时关闭数据收集
- ✅ **不出售数据** - 我们绝不出售或共享您的个人信息

## 🔐 我们收集的信息

### 1. 不收集的信息

Renomee **不会收集**以下信息：

- ❌ 您的文件内容
- ❌ 文件名和路径
- ❌ 个人身份信息（姓名、地址、电话）
- ❌ 支付信息（由第三方支付处理）
- ❌ 浏览历史
- ❌ 位置信息

### 2. 自动收集的信息

为了改进产品，我们会收集以下匿名统计数据：

#### 应用使用数据

```json
{
  "app_version": "1.2.0",
  "os": "Windows 11",
  "launch_count": 15,
  "feature_usage": {
    "ai_mode": 3,
    "regex_mode": 1,
    "script_mode": 0
  }
}
```

- 应用版本号
- 操作系统类型和版本
- 应用启动次数
- 功能使用频率（如 AI 模式使用次数）
- 错误和崩溃报告

**⚙️ 控制方式：**

设置 → 隐私 → 关闭 "使用数据收集"

#### 性能数据

- 重命名操作耗时
- 处理文件数量统计
- 内存和 CPU 使用情况

这些数据用于性能优化，完全匿名。

### 3. 可选收集的信息

以下信息只在您主动提供或开启功能时收集：

#### AI 模式数据

使用 AI 智能识别功能时：

- **图片内容识别** - 上传缩略图（< 100KB）
- **文档关键词提取** - 上传文本摘要（< 10KB）

**保护措施：**

- 数据加密传输（TLS 1.3）
- 处理完成后立即删除
- 不保存任何文件
- 可在设置中完全禁用

#### 反馈和支持

当您联系技术支持或提交反馈时：

- 邮箱地址
- 您提供的日志文件
- 问题描述和截图

**用途：** 仅用于回复您的咨询和改进产品。

### 4. 本地存储的信息

以下信息仅保存在您的设备上，不上传：

- 应用配置和设置
- 保存的预设
- 操作历史记录
- 自定义元数据

**存储位置：**

- Windows: `%APPDATA%\Renomee\`
- macOS: `~/Library/Application Support/Renomee/`

您可以随时删除这些文件。

## 🔒 信息保护措施

我们采取以下措施保护您的信息：

### 技术措施

- 🔐 **加密传输** - 所有网络请求使用 TLS 1.3 加密
- 🛡️ **安全存储** - 敏感配置使用 AES-256 加密
- 🚫 **最小权限** - 只请求必要的系统权限
- 🔍 **代码审计** - 定期安全审计和渗透测试

### 管理措施

- 👥 **访问控制** - 严格限制数据访问权限
- 📋 **隐私培训** - 团队成员隐私保护培训
- 🔄 **定期审查** - 定期审查数据收集和使用情况
- 📝 **透明报告** - 每年发布透明度报告

### 2. 更正权

如果数据不准确，您可以要求更正：

- 联系 support@xiaojingjia.top

### 3. 删除权

您可以要求删除您的数据：

- 设置 → 隐私 → 关闭所有数据收集

### 5. 数据可携带

您可以导出您的数据：

- 设置 → 隐私 → 导出我的数据

### 6. 反对权

您可以反对特定的数据处理：

- 联系 privacy@xiaojingjia.top 说明您的诉求
