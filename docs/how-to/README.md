## ✅ SEO 检查清单

每篇文章发布前必须确认：

### 内容质量
- [ ] 字数: 1,500-3,000 字
- [ ] 主关键词出现在: 标题、H1、首段、至少 1 个 H2、结论
- [ ] 副关键词自然分布
- [ ] 3-5 句段落，避免大段文字
- [ ] 每 500 字至少 1 个外部引用

### 结构
- [ ] H1 只有 1 个
- [ ] H2/H3 层级清晰
- [ ] 快速导航锚点
- [ ] FAQ 部分（7+ 问题）
- [ ] 总结 + CTA

### 链接
- [ ] 3-5 个内链（指向相关内容）
- [ ] 2-3 个外链（权威来源）
- [ ] 所有链接都有描述性锚文本

### 多媒体
- [ ] 至少 3 张截图
- [ ] 所有图片都有 alt 文本
- [ ] 表格用于对比信息
- [ ] 提示框突出重点

### 技术 SEO
- [ ] Meta description 150-160 字符
- [ ] URL slug 简洁（kebab-case）
- [ ] 关键词密度 1-2%
- [ ] 移动端友好（表格不超过 4 列）
