# 使用场景

Renomee AI 不仅仅是一个重命名工具，更是你的智能文件管理助手。以下是用户最常用的应用场景。

---

## 🎯 热门场景

### 🗂️ [下载文件整理](downloads/)
**痛点**：下载文件夹混乱不堪，找文件像大海捞针

**解决方案**：
- 自动识别文件类型并分类
- 根据内容生成描述性文件名
- 批量移除无意义的后缀（如 `_downloaded`, `(1)`, `final_v3`）

**效果**：从混乱到井然有序，找文件只需 3 秒

**即将推出的指南**：如何自动整理下载文件夹

---

### 🎓 [学术研究与论文管理](research/)
**痛点**：数百篇论文文件名是随机 ID，根本不知道哪篇是哪篇

**解决方案**：
- 从 PDF 中提取完整标题
- 自动识别作者和发表年份
- 支持 OCR，处理扫描版论文
- 批量处理 arXiv、PubMed、IEEE 下载的文件

**效果**：127 篇论文从混乱 ID 变为 `标题_作者_年份.pdf` 格式

**相关资源**：
- [如何根据标题重命名 PDF](../how-to/rename-pdf-by-title.md)
- [学术论文批量重命名案例](https://renomeeai.com/blog/pdf-batch-rename-scholar/)

---

### 💼 [商务文档管理](business/)
**痛点**：发票、合同、报告散落各处，归档耗时费力

**解决方案**：
- **发票**：OCR 识别供应商、日期、金额
- **合同**：提取合同方名称和签署日期
- **报告**：按标题和版本号规范命名

**效果**：财务归档时间从数小时缩短到几分钟

**适用场景**：
- 财务部门归档发票收据
- 法务部门管理合同文档
- 行政部门整理会议纪要

**即将推出的指南**：
- 如何自动归档发票
- 如何整理合同文档

---

### 📸 [媒体文件管理](media/)
**痛点**：照片、截图、视频文件名毫无意义，难以查找

**解决方案**：
- **照片**：按 EXIF 数据重命名（日期、地点、相机）
- **截图**：OCR 识别内容或按应用名称分类
- **视频**：提取元数据，按分辨率和时长命名

**效果**：数千张照片瞬间变得井井有条

**适用场景**：
- 摄影师整理作品集
- 设计师管理素材库
- 个人用户整理手机照片
- 自媒体创作者管理视频素材

**即将推出的指南**：
- 如何按日期批量重命名照片
- 如何整理截图文件
- 如何管理视频素材库
