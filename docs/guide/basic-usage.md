## 📂 添加文件

Renomee 支持多种方式添加文件。

### 方式 1：拖放文件

这是最快捷的方式：

1. 从文件管理器选择文件或文件夹
2. 拖动到 Renomee 窗口
3. 松开鼠标，文件自动加载

!!! tip "批量拖放"
    可以同时拖放多个文件夹和文件，Renomee 会自动合并列表。

### 方式 2：选择文件夹

点击 **"选择文件夹"** 按钮：

- 浏览并选择文件夹
- 可选择是否包含子文件夹
- 可选择要包含的文件类型

!!! example "包含子文件夹"
    ```
    📁 Photos/
       ├── 2024/
       │   ├── photo1.jpg
       │   └── photo2.jpg
       └── 2025/
           └── photo3.jpg
    ```
    勾选 "包含子文件夹" 会处理所有 3 个文件。

### 方式 3：选择文件

点击 **"选择文件"** 按钮：

- 可按住 `Ctrl` (Windows) 或 `Cmd` (macOS) 多选
- 可按住 `Shift` 连续选择
- 支持预览文件信息

### 文件过滤

添加文件后，可以使用过滤功能：

=== "按文件类型"
    ```
    ☑ 图片 (.jpg, .png, .gif)
    ☐ 文档 (.pdf, .docx, .txt)
    ☑ 视频 (.mp4, .avi, .mov)
    ```

=== "按文件大小"
    ```
    最小: 100 KB
    最大: 10 MB
    ```

=== "按修改时间"
    ```
    从: 2025-01-01
    到: 2025-01-31
    ```

## 🗣️ 自然语言规则

Renomee 最强大的功能是支持自然语言描述重命名规则。

### 基础操作

#### 添加前缀

```
添加前缀 "备份_"
```

**结果：**
```
photo.jpg       →  备份_photo.jpg
document.pdf    →  备份_document.pdf
```

#### 添加后缀

```
在扩展名前添加 "_副本"
```

**结果：**
```
photo.jpg       →  photo_副本.jpg
document.pdf    →  document_副本.pdf
```

#### 替换文本

```
把 "IMG" 替换成 "Photo"
```

**结果：**
```
IMG_001.jpg     →  Photo_001.jpg
IMG_002.jpg     →  Photo_002.jpg
```

#### 删除文本

```
删除文件名中的 "副本"
```

**结果：**
```
photo_副本.jpg   →  photo.jpg
file_副本.pdf    →  file.pdf
```

#### 转换大小写

```
文件名转小写
```

**结果：**
```
PHOTO.JPG       →  photo.jpg
MyFile.PDF      →  myfile.pdf
```

### 组合操作

可以用逗号或顿号连接多个操作：

```
删除 "IMG"，转小写，添加前缀 "photo_"
```

**结果：**
```
IMG_001.JPG     →  photo_001.jpg
IMG_002.JPG     →  photo_002.jpg
```

## 🔢 使用变量

变量可以插入动态内容到文件名中。

### 常用变量

| 变量 | 说明 | 示例输出 |
|------|------|----------|
| `{name}` | 原文件名（不含扩展名） | photo |
| `{ext}` | 文件扩展名 | .jpg |
| `{n}` | 序号（从1开始） | 001 |
| `{date}` | 修改日期 | 2025-01-26 |
| `{time}` | 修改时间 | 14-30-00 |
| `{parent}` | 父文件夹名 | Photos |
| `{size}` | 文件大小 | 1.2MB |

### 日期时间变量

| 变量 | 格式 | 示例 |
|------|------|------|
| `{date}` | YYYY-MM-DD | 2025-01-26 |
| `{date:YYYYMMDD}` | 自定义格式 | 20250126 |
| `{date:YYYY年MM月DD日}` | 中文格式 | 2025年01月26日 |
| `{time}` | HH-MM-SS | 14-30-00 |
| `{time:HHMMSS}` | 自定义格式 | 143000 |

!!! example "日期变量示例"
    规则：`{date}_{name}{ext}`
    
    **结果：**
    ```
    photo.jpg       →  2025-01-26_photo.jpg
    document.pdf    →  2025-01-26_document.pdf
    ```

### 序号变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `{n}` | 1位序号 | 1, 2, 3, ... |
| `{n2}` | 2位序号（补零） | 01, 02, 03, ... |
| `{n3}` | 3位序号（补零） | 001, 002, 003, ... |
| `{n4}` | 4位序号（补零） | 0001, 0002, 0003, ... |

!!! example "序号变量示例"
    规则：`Photo_{n3}{ext}`
    
    **结果：**
    ```
    IMG_1234.jpg    →  Photo_001.jpg
    IMG_5678.jpg    →  Photo_002.jpg
    ```

### 组合变量

```
{parent}_{date}_{n3}_{name}{ext}
```

**结果：**
```
photo.jpg       →  Photos_2025-01-26_001_photo.jpg
```

## 🤖 AI 智能模式

启用 AI 模式后，Renomee 会智能分析文件内容。

### 启用 AI 模式

1. 点击界面上的 **"🤖 AI模式"** 按钮
2. 或在规则中添加 `[AI]` 标记

### 图片文件识别

AI 可以从图片中提取：

- 📅 **拍摄日期和时间**
- 📍 **拍摄地点**（如果有 GPS 信息）
- 📷 **相机型号和参数**
- 🎨 **图片主题**（风景、人像、动物等）

!!! example "AI 图片重命名"
    规则：`[AI] 按内容智能命名`
    
    **结果：**
    ```
    IMG_1234.jpg    →  2025-01-26_上海外滩_日落.jpg
    IMG_5678.jpg    →  2025-01-26_人像_微笑.jpg
    ```

### 文档文件识别

AI 可以从文档中提取：

- 📄 **文档标题**
- ✍️ **作者信息**
- 🔑 **关键词**
- 📅 **创建/修改日期**

### 音视频文件识别

AI 可以从音视频中提取：

- 🎵 **歌曲名称**
- 🎤 **艺术家/演唱者**
- 💿 **专辑名称**
- 🎬 **视频标题**
- ⏱️ **时长信息**

## 👁️ 预览功能

执行重命名前，建议先预览结果。

### 预览模式

点击 **"预览"** 按钮后：

```
┌─────────────────────────────────────────────┐
│  预览重命名结果 (10 个文件)                │
├─────────────────────────────────────────────┤
│  原文件名              →  新文件名          │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━  │
│  ✓ IMG_0001.jpg       →  Photo_001.jpg     │
│  ✓ IMG_0002.jpg       →  Photo_002.jpg     │
│  ⚠ IMG_0003.jpg       →  Photo_003.jpg     │
│     (文件名已存在)                          │
│  ✓ IMG_0004.jpg       →  Photo_004.jpg     │

```

### 状态图标说明

| 图标 | 含义 | 说明 |
|------|------|------|
| ✓ | 成功 | 可以安全重命名 |
| ⚠ | 警告 | 可能存在问题（如文件名冲突） |
| ✗ | 错误 | 无法重命名（如文件被占用） |
| ℹ | 信息 | 文件将被跳过 |

### 冲突处理

如果出现文件名冲突，Renomee 提供以下选项：

=== "自动编号"
    ```
    Photo_001.jpg  →  Photo_001(1).jpg
    Photo_001.jpg  →  Photo_001(2).jpg
    ```

=== "覆盖原文件"
    ```
    ⚠ 警告：此操作不可撤销！
    ```

=== "跳过冲突"
    ```
    保持原文件名不变
    ```

## ↩️ 撤销操作

如果重命名结果不理想，可以一键撤销。

### 立即撤销

在完成报告界面点击 **"撤销"** 按钮，所有更改会立即恢复。

### 历史记录撤销

点击 **菜单** → **历史记录**：

```
┌─────────────────────────────────────────────┐
│  操作历史                                  │
├─────────────────────────────────────────────┤
│  📅 2025-01-26 14:30                       │
│  规则: {date}_{n3}{ext}                    │
│  文件: 100 个                              │
│  [撤销] [查看详情]                         │
│  ─────────────────────────────────────────  │
│  📅 2025-01-26 10:15                       │
│  规则: 添加前缀 "Photo_"                  │
│  文件: 50 个                               │
│  [撤销] [查看详情]                         │
└─────────────────────────────────────────────┘
```

!!! warning "撤销限制"
    - 只能撤销最近 10 次操作
    - 如果原文件已被删除或移动，无法撤销
    - 建议重要文件先备份

## 💾 保存配置

经常使用的规则可以保存为预设。

### 保存预设

1. 输入规则后，点击规则框右侧的 **"⭐ 保存"** 按钮
2. 输入预设名称，如 "照片按日期命名"
3. 点击 **"确定"** 保存

### 使用预设

点击规则框右侧的 **"📁 预设"** 按钮，选择已保存的预设：

```
我的预设:
- 📸 照片按日期命名
- 📄 文档添加作者前缀
- 🎵 音乐文件标准化
- 🎬 视频按分辨率分类
```

### 导入导出

可以导出预设分享给其他人：

1. **导出:** 菜单 → 预设 → 导出 → 保存为 `.renomee` 文件
2. **导入:** 菜单 → 预设 → 导入 → 选择 `.renomee` 文件

## ⚙️ 设置

点击右上角的 **"⚙"** 图标打开设置。

### 常规设置

- **语言:** 简体中文 / English
- **主题:** 浅色 / 深色 / 跟随系统
- **启动时:** 打开上次的文件列表

### 重命名设置

- **默认序号起始:** 1 / 0 / 自定义
- **序号位数:** 2 / 3 / 4
- **冲突处理:** 自动编号 / 跳过 / 询问
- **大小写:** 保持原样 / 全部小写 / 全部大写

### AI 设置

- **AI 模式:** 启用 / 禁用
- **识别语言:** 中文 / 英文 / 自动检测
- **识别精度:** 快速 / 标准 / 精确

### 高级设置

- **日志级别:** 错误 / 警告 / 信息 / 调试
- **历史记录保留:** 10 / 20 / 50 条
- **备份设置:** 自动备份原文件名
