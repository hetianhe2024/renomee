## 📂 添加文件

Renomee 支持多种方式添加文件。

### 方式 1：拖放文件

这是最快捷的方式：

1. 从文件管理器选择文件或文件夹
2. 拖动到 Renomee 窗口
3. 松开鼠标，文件自动加载

!!! tip "批量拖放"
    可以同时拖放多个文件夹和文件，Renomee 会自动合并列表。

### 方式 2：选择文件夹

点击 **"选择文件夹"** 按钮：

- 浏览并选择文件夹
- 可选择是否包含子文件夹
- 可选择要包含的文件类型

!!! example "包含子文件夹"
    ```
    📁 Photos/
       ├── 2024/
       │   ├── photo1.jpg
       │   └── photo2.jpg
       └── 2025/
           └── photo3.jpg
    ```
    勾选 "包含子文件夹" 会处理所有 3 个文件。

### 方式 3：选择文件

点击 **"选择文件"** 按钮：

- 可按住 `Ctrl` (Windows) 或 `Cmd` (macOS) 多选
- 可按住 `Shift` 连续选择
- 支持预览文件信息

### 文件过滤

添加文件后，可以使用过滤功能：

=== "按文件类型"
    ```
    ☑ 图片 (.jpg, .png, .gif)
    ☐ 文档 (.pdf, .docx, .txt)
    ☑ 视频 (.mp4, .avi, .mov)
    ```

=== "按文件大小"
    ```
    最小: 100 KB
    最大: 10 MB
    ```

=== "按修改时间"
    ```
    从: 2025-01-01
    到: 2025-01-31
    ```

## 🗣️ 自然语言规则

Renomee 最强大的功能是支持自然语言描述重命名规则。

### 基础操作

#### 添加前缀

```
添加前缀 "备份_"
```

**结果：**
```
photo.jpg       →  备份_photo.jpg
document.pdf    →  备份_document.pdf
```

#### 添加后缀

```
在扩展名前添加 "_副本"
```

**结果：**
```
photo.jpg       →  photo_副本.jpg
document.pdf    →  document_副本.pdf
```

#### 替换文本

```
把 "IMG" 替换成 "Photo"
```

**结果：**
```
IMG_001.jpg     →  Photo_001.jpg
IMG_002.jpg     →  Photo_002.jpg
```

#### 删除文本

```
删除文件名中的 "副本"
```

**结果：**
```
photo_副本.jpg   →  photo.jpg
file_副本.pdf    →  file.pdf
```

#### 转换大小写

```
文件名转小写
```

**结果：**
```
PHOTO.JPG       →  photo.jpg
MyFile.PDF      →  myfile.pdf
```

### 组合操作

可以用逗号或顿号连接多个操作：

```
删除 "IMG"，转小写，添加前缀 "photo_"
```

**结果：**
```
IMG_001.JPG     →  photo_001.jpg
IMG_002.JPG     →  photo_002.jpg
```

## 🔢 使用变量

变量可以插入动态内容到文件名中。

### 常用变量

| 变量 | 说明 | 示例输出 |
|------|------|----------|
| `{name}` | 原文件名（不含扩展名） | photo |
| `{ext}` | 文件扩展名 | .jpg |
| `{n}` | 序号（从1开始） | 001 |
| `{date}` | 修改日期 | 2025-01-26 |
| `{time}` | 修改时间 | 14-30-00 |
| `{parent}` | 父文件夹名 | Photos |
| `{size}` | 文件大小 | 1.2MB |

### 日期时间变量

| 变量 | 格式 | 示例 |
|------|------|------|
| `{date}` | YYYY-MM-DD | 2025-01-26 |
| `{date:YYYYMMDD}` | 自定义格式 | 20250126 |
| `{date:YYYY年MM月DD日}` | 中文格式 | 2025年01月26日 |
| `{time}` | HH-MM-SS | 14-30-00 |
| `{time:HHMMSS}` | 自定义格式 | 143000 |

!!! example "日期变量示例"
    规则：`{date}_{name}{ext}`
    
    **结果：**
    ```
    photo.jpg       →  2025-01-26_photo.jpg
    document.pdf    →  2025-01-26_document.pdf
    ```

### 序号变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `{n}` | 1位序号 | 1, 2, 3, ... |
| `{n2}` | 2位序号（补零） | 01, 02, 03, ... |
| `{n3}` | 3位序号（补零） | 001, 002, 003, ... |
| `{n4}` | 4位序号（补零） | 0001, 0002, 0003, ... |

!!! example "序号变量示例"
    规则：`Photo_{n3}{ext}`
    
    **结果：**
    ```
    IMG_1234.jpg    →  Photo_001.jpg
    IMG_5678.jpg    →  Photo_002.jpg
    ```

### 组合变量

```
{parent}_{date}_{n3}_{name}{ext}
```

**结果：**
```
photo.jpg       →  Photos_2025-01-26_001_photo.jpg
```

## 🤖 AI 智能模式

启用 AI 模式后，Renomee 会智能分析文件内容，支持大文件和 OCR 识别。

### 启用 AI 模式

1. 点击界面上的 **"🤖 AI模式"** 按钮
2. 或在规则中添加 `[AI]` 标记
3. AI 会根据需要自动提取文件内容

### 文档文件识别（支持大文件）

AI 可以从 PDF、Word、Excel 中提取内容（支持 50MB 以内大文件）：

#### PDF 文档（文字版 + 扫描版）

- 📄 **文档标题** - PDF 元数据或首行智能提取
- ✍️ **作者信息** - PDF 元数据中的作者
- 📝 **正文内容** - 前 1000 字符预览
- 📑 **章节标题** - 自动识别的标题列表
- 🖼️ **OCR 智能兜底** - 扫描版 PDF、图形页自动识别文字

!!! tip "OCR 兜底机制"
    当 PDF 页面文字稀少（如封面、图表页）时，Renomee 会自动启用 OCR 识别，确保内容不遗漏。

!!! example "PDF 智能重命名"
    规则：`根据 PDF 标题重命名`
    
    **结果：**
    ```
    document1.pdf        →  深度学习在图像识别中的应用研究_张三_2024.pdf
    scan001.pdf          →  购销合同_甲方乙方_20240316.pdf（OCR识别）
    report_large.pdf     →  2024年度财务报告_完整版_158页.pdf（30MB大文件）
    ```

#### Word 文档

- 📄 **文档标题** - Word 文档属性中的标题
- 📝 **正文预览** - 文档正文前 1000 字符
- 📑 **章节标题** - 自动识别的标题层级列表

#### Excel / CSV

- 📊 **工作表名称** - 所有工作表名称列表
- 📋 **列标题** - 第一行内容
- 🔢 **总行数** - 数据行数统计

### 图片文件识别（EXIF + OCR）

AI 可以从图片中提取元数据和文字内容：

#### EXIF 元数据提取

- 📅 **拍摄日期和时间** - 相机快门按下的时刻
- 📍 **拍摄地点** - GPS 位置信息
- 📷 **相机型号和参数** - 品牌、型号、光圈、快门、ISO
- 🖼️ **图片分辨率** - 宽度和高度

#### OCR 文字识别 🔥

- 📸 **截图识别** - 自动提取截图中的文字内容
- 🪪 **证件照识别** - 身份证、护照等证件信息提取
- 💳 **名片识别** - 姓名、职位、公司等信息
- 📋 **菜单/海报识别** - 图片中的文字内容

!!! example "图片 OCR 重命名"
    规则：`识别截图内的文字命名`
    
    **结果：**
    ```
    IMG_1234.jpg         →  2025-01-26_上海外滩_日落.jpg（EXIF）
    screenshot.png       →  登录页面_用户名密码_20240316.png（OCR）
    namecard.jpg         →  张三_产品经理_某公司_名片.jpg（OCR）
    idcard_scan.jpg      →  身份证_张三_20240316.jpg（OCR）
    ```

### 音视频文件识别

AI 可以从音视频中提取标签信息：

#### 音频文件

- 🎵 **歌曲名称** - 曲目标题
- 🎤 **艺术家/演唱者** - 歌手或乐队
- 💿 **专辑名称** - 所属专辑
- 📅 **发行年份** - 发行时间
- 🎼 **流派** - 音乐风格
- ⏱️ **时长信息** - 播放时长

#### 视频文件

- 🎬 **视频标题** - 视频文件标题标签
- 📐 **分辨率** - 1080p、4K 等
- ⏱️ **时长** - 视频长度
- 🎞️ **帧率** - 30fps、60fps 等
- 🗜️ **编码格式** - H.264、H.265、AV1 等

!!! example "音视频智能重命名"
    规则：`按艺术家-歌曲名格式重命名`
    
    **结果：**
    ```
    track01.mp3          →  周杰伦 - 晴天 (2001).mp3
    video001.mp4         →  会议录像_1080p_H265_90min.mp4
    ```

### 状态图标说明

| 图标 | 含义 | 说明 |
|------|------|------|
| ✓ | 成功 | 可以安全重命名 |
| ⚠ | 警告 | 可能存在问题（如文件名冲突） |
| ✗ | 错误 | 无法重命名（如文件被占用） |
| ℹ | 信息 | 文件将被跳过 |

### 冲突处理

如果出现文件名冲突，Renomee 提供以下选项：

=== "自动编号"
    ```
    Photo_001.jpg  →  Photo_001(1).jpg
    Photo_001.jpg  →  Photo_001(2).jpg
    ```

=== "覆盖原文件"
    ```
    ⚠ 警告：此操作不可撤销！
    ```

=== "跳过冲突"
    ```
    保持原文件名不变
    ```

## ↩️ 撤销操作

如果重命名结果不理想，可以一键撤销。

### 立即撤销

在完成报告界面点击 **"撤销"** 按钮，所有更改会立即恢复。

### 历史记录撤销

点击 **菜单** → **历史记录**：

```
┌─────────────────────────────────────────────┐
│  操作历史                                  │
├─────────────────────────────────────────────┤
│  📅 2025-01-26 14:30                       │
│  规则: {date}_{n3}{ext}                    │
│  文件: 100 个                              │
│  [撤销] [查看详情]                         │
│  ─────────────────────────────────────────  │
│  📅 2025-01-26 10:15                       │
│  规则: 添加前缀 "Photo_"                  │
│  文件: 50 个                               │
│  [撤销] [查看详情]                         │
└─────────────────────────────────────────────┘
```

!!! warning "撤销限制"
    - 只能撤销最近 10 次操作
    - 如果原文件已被删除或移动，无法撤销
    - 建议重要文件先备份

## 💾 保存配置

经常使用的规则可以保存为预设。

### 保存预设

1. 输入规则后，点击规则框右侧的 **"⭐ 保存"** 按钮
2. 输入预设名称，如 "照片按日期命名"
3. 点击 **"确定"** 保存

### 使用预设

点击规则框右侧的 **"📁 预设"** 按钮，选择已保存的预设：

```
我的预设:
- 📸 照片按日期命名
- 📄 文档添加作者前缀
- 🎵 音乐文件标准化
- 🎬 视频按分辨率分类
```

### 导入导出

可以导出预设分享给其他人：

1. **导出:** 菜单 → 预设 → 导出 → 保存为 `.renomee` 文件
2. **导入:** 菜单 → 预设 → 导入 → 选择 `.renomee` 文件
