# Renomee AI - 智能文件重命名工具

![Renomee Logo](https://renomeeai.com//static/media/logo.png)

欢迎使用 **Renomee AI**，这是一款革命性的智能文件重命名工具，让文件管理变得简单高效！

## 🎯 为什么选择 Renomee？

传统的文件重命名工具需要学习复杂的正则表达式，而 Renomee AI 让一切变得简单：

- 🤖 **AI 智能识别** - 自动识别文件内容，生成有意义的文件名
- 🗣️ **自然语言操作** - 用说话的方式描述规则，无需学习正则表达式
- ⚡ **批量高效处理** - 一次处理数百个文件，效率提升 10 倍
- 🔒 **数据安全可靠** - 本地处理，文件不上传，隐私有保障
- 📱 **跨平台支持** - 支持 Windows 10+ 和 macOS 10.15+

## 🚀 快速开始

### 安装 Renomee

1. 访问 [官方网站](https://renomeeai.com/) 下载安装包
2. 运行安装程序，按照向导完成安装
3. 启动 Renomee，开始您的智能重命名之旅

### 第一次重命名

```bash
# 示例：将照片按日期重命名
选择文件夹 → 输入规则："按拍摄日期命名" → 点击执行
```

只需三步，轻松完成！详见 [快速开始指南](quickstart.md)。

## 📚 核心功能

### 1. AI 智能识别

Renomee 能自动识别：

- 📸 **图片文件** - 提取拍摄日期、地点、设备信息
- 📄 **文档文件** - 识别文档标题、作者、创建时间
- 🎵 **音频文件** - 提取歌曲名称、歌手、专辑信息
- 🎬 **视频文件** - 识别视频标题、分辨率、时长

### 2. 自然语言规则

无需学习复杂语法，直接用中文描述：

!!! example "自然语言示例"
    - "把所有 jpg 图片改成日期格式"
    - "将文件名中的空格替换为下划线"
    - "给所有视频添加前缀 VID"
    - "按创建时间从 1 开始编号"

### 3. 批量处理

支持同时处理：

- ✅ 多个文件夹
- ✅ 不同文件类型
- ✅ 嵌套子目录
- ✅ 预览重命名结果

### 4. 安全保障

- 🔍 **预览模式** - 执行前查看所有更改
- ↩️ **一键撤销** - 随时恢复原始文件名
- 📝 **操作日志** - 完整记录每次操作
- 🛡️ **本地处理** - 文件不上传，完全离线

## 💡 使用场景

### 摄影师整理照片

将数千张照片按日期、地点、事件自动分类命名：

```
IMG_1234.jpg → 2025-01-26_上海外滩_日落_001.jpg
```

### 视频创作者管理素材

统一命名规范，快速定位素材：

```
VID00023.mp4 → 【教程】PS技巧-第05集-图层混合.mp4
```

### 开发者规范文件名

确保代码仓库命名一致：

```
新建文件1.js → user-profile-component.js
```
