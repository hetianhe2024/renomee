# Renomee AI - 智能文件重命名工具

![Renomee Logo](https://renomeeai.com//static/media/logo.png)

欢迎使用 **Renomee AI**，这是一款革命性的智能文件重命名工具，让文件管理变得简单高效！

## 🎯 为什么选择 Renomee？

传统的文件重命名工具需要学习复杂的正则表达式，而 Renomee AI 让一切变得简单：

- 🤖 **AI 智能识别** - 自动识别文件内容，生成有意义的文件名
- 🗣️ **自然语言操作** - 用说话的方式描述规则，无需学习正则表达式
- 🧠 **内容理解能力** - 不只看文件名，能读懂文件内部内容（支持 50MB 大文件）
- 🖼️ **OCR 智能识别** - 扫描版 PDF、图形页、截图、证件照自动识别文字
- ⚡ **批量处理优化** - 队列化架构，处理数百文件不卡顿，实时进度反馈
- 🔒 **数据安全可靠** - 本地处理，文件不上传，隐私有保障
- 📱 **跨平台支持** - 支持 Windows 10+ 和 macOS 10.15+

## 🚀 快速开始

### 安装 Renomee

1. 访问 [Renomee AI 智能文件重命名工具](https://renomeeai.com/) 下载安装包
2. 运行安装程序，按照向导完成安装
3. 启动 Renomee，开始您的智能重命名之旅

### 第一次重命名

```bash
# 示例：将照片按日期重命名
选择文件夹 → 输入规则："按拍摄日期命名" → 点击执行
```

只需三步，轻松完成！详见 [快速开始指南](quickstart.md)。

## 📚 核心功能

### 1. AI 智能识别与内容提取

Renomee 不仅能识别文件元数据，还能深度理解文件内容：

- 📄 **PDF 文档** - 提取标题、作者、正文内容（支持 50MB 大文件）
  - 🖼️ **OCR 智能兜底** - 扫描版 PDF、图形页自动识别文字
- 📝 **Word/Excel** - 识别文档标题、章节、工作表名（支持 50MB 大文件）
- 📸 **图片文件** - 提取拍摄日期、地点、设备信息
  - 🖼️ **OCR 文字识别** - 截图、证件照、名片自动提取文字内容
- 🎵 **音频文件** - 提取歌曲名称、歌手、专辑信息
- 🎬 **视频文件** - 识别视频标题、分辨率、时长

!!! tip "智能内容感知"
    Renomee 采用内容感知机制，基于文件真实内容生成精准命名，而非简单的模板套用。例如：
    
    - 论文 PDF → `深度学习在图像识别中的应用研究_张三_2024.pdf`
    - 扫描合同 → `购销合同_甲方乙方_20240316.pdf`（OCR识别）
    - 名片截图 → `张三_产品经理_某公司_名片.jpg`（OCR识别）

### 2. 自然语言规则

无需学习复杂语法，直接用中文描述：

!!! example "自然语言示例"
    - "把所有 jpg 图片改成日期格式"
    - "将文件名中的空格替换为下划线"
    - "给所有视频添加前缀 VID"
    - "按创建时间从 1 开始编号"

### 3. 批量处理与大文件支持

采用队列化架构，确保大规模处理的流畅体验：

- ✅ **批量文件支持** - 一次处理数百个文件，无需逐个操作
- ✅ **大文件处理优化** - 支持 50MB 以内的 PDF、Word、Excel 等大文件内容提取
- ✅ **实时进度反馈** - 批量操作时显示详细进度条和处理状态
- ✅ **智能队列管理** - 内容提取请求自动排队，避免系统卡顿
- ✅ **多个文件夹** - 同时处理不同来源的文件
- ✅ **嵌套子目录** - 递归处理文件夹结构
- ✅ **预览重命名结果** - 执行前查看所有更改

!!! example "批量处理性能"
    - **小文件批量重命名** - 1000+ 个文件，< 5 秒
    - **中等文件批量提取** - 50 个 5MB 文件，30-60 秒
    - **大文件批量提取** - 20 个 30-50MB 文件，60-120 秒

### 4. 安全保障

- 🔍 **预览模式** - 执行前查看所有更改
- ↩️ **一键撤销** - 随时恢复原始文件名
- 📝 **操作日志** - 完整记录每次操作

## 💡 使用场景

### 学术研究者整理文献

一次性整理 500+ 篇论文 PDF（包含大文件和扫描版）：

```
document1.pdf → 深度学习在图像识别中的应用研究_张三_2024.pdf
scan001.pdf → 计算机视觉综述_李四_2023.pdf（OCR识别）
```

**效果：** 从数小时手动整理缩短到 5 分钟批量处理

### 摄影师整理照片

将数千张照片按日期、地点、事件自动分类命名：

```
IMG_1234.jpg → 2025-01-26_上海外滩_日落_001.jpg
screenshot.png → 登录页面_用户名密码_20240316.png（OCR识别）
```

### 财务人员归档票据

批量处理扫描版发票和合同（OCR自动识别内容）：

```
scan001.pdf → 购销合同_甲方乙方_20240316.pdf
invoice.jpg → 发票_供应商名称_2024-03-16_1258元.jpg
```

**效果：** 月底对账时间从 1 天缩短到 2 小时

### 视频创作者管理素材

统一命名规范，快速定位素材：

```
VID00023.mp4 → 【教程】PS技巧-第05集-图层混合_1080p_H265.mp4
```

### 开发者规范文件名

确保代码仓库命名一致：

```
新建文件1.js → user-profile-component.js
```
