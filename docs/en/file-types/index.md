# Rename by File Type

Different file types contain different information. Renomee AI recognizes various file format characteristics and extracts the most valuable information to generate meaningful filenames.

---

## 📑 Supported File Types

### 📄 [PDF Files](pdf/)
**Use Cases**: Academic papers, ebooks, reports, contracts, invoices

**Smart Extraction**:
- Document title (from metadata or content)
- Author information
- Creation date
- OCR text recognition (supports scanned PDFs)

**Popular Guides**:
- [How to Rename PDFs by Title](../../how-to/rename-pdf-by-title.md)

**File Size Support**: Up to 50MB

---

### 📸 [Images & Photos](images/)
**Use Cases**: Personal photos, product images, design files, screenshots

**Smart Extraction**:
- EXIF data (date taken, location, camera model)
- Text in images (OCR)
- Image dimensions and format info

**Supported Formats**: JPG, PNG, GIF, BMP, TIFF, HEIC, WebP

---

### 📝 [Office Documents](documents/)
**Use Cases**: Word docs, Excel spreadsheets, PowerPoint presentations

**Smart Extraction**:
- Document title and subject
- Author information
- Modification date
- Worksheet names (Excel)
- Slide titles (PowerPoint)

**Supported Formats**: .docx/.doc, .xlsx/.xls, .pptx/.ppt

---

### 🎵 [Audio Files](audio/)
**Use Cases**: Music library, podcasts, recordings, audiobooks

**Smart Extraction**:
- ID3 tags (artist, album, title)
- Bitrate and duration
- Year and genre

**Supported Formats**: MP3, M4A, WAV, FLAC, AAC, OGG

---

### 🎬 [Video Files](video/)
**Use Cases**: Video footage, screen recordings, movies, tutorial videos

**Smart Extraction**:
- Video metadata (title, description)
- Resolution and codec info
- Duration and framerate

**Supported Formats**: MP4, AVI, MOV, MKV, WMV, FLV
