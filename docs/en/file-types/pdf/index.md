# PDF File Renaming

PDF is one of the most common document formats. Renomee AI provides powerful PDF content recognition capabilities to help you quickly organize large volumes of documents.

## 🎯 Core Features

- **Title extraction**: Auto-identify document titles
- **OCR support**: Process scanned PDFs
- **Metadata reading**: Extract author, date, and other info
- **Large file optimization**: Supports PDFs up to 50MB
