## 🎯 Popular Scenarios

### 🎓 [Academic Research & Paper Management](research/)
**Pain Point**: Hundreds of papers with random IDs—impossible to tell which is which

**Solution**:
- Extract complete titles from PDFs
- Auto-identify authors and publication years
- Support OCR for scanned papers
- Batch process files from arXiv, PubMed, IEEE

**Impact**: 127 papers transformed from random IDs to `Title_Author_Year.pdf` format

### 🗂️ [Downloads Folder Organization](downloads/)
**Pain Point**: Downloads folder is a mess—finding files is like finding a needle in a haystack

**Solution**:
- Auto-identify file types and classify
- Generate descriptive filenames based on content
- Batch remove meaningless suffixes (like `_downloaded`, `(1)`, `final_v3`)

**Impact**: From chaos to order—finding files takes 3 seconds

**Pain Point**: Invoices, contracts, reports scattered everywhere—archiving is time-consuming

**Solution**:

- **Contracts**: Extract party names and signing dates
- **Reports**: Standardize naming by title and version

**Impact**: Financial archiving time reduced from hours to minutes

### 📸 [Media File Management](media/)
**Pain Point**: Photos, screenshots, videos have meaningless names—hard to find

**Solution**:
- **Photos**: Rename by EXIF data (date, location, camera)
- **Screenshots**: OCR content or classify by app name
- **Videos**: Extract metadata, name by resolution and duration
