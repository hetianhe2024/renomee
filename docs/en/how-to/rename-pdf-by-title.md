## Troubleshooting Common Issues

### Problem: Title Not Detected

**Cause**: PDF has no clear title structure or is entirely image-based

**Solutions**:
- Enable OCR processing (Renomee does this automatically)
- Manually verify 1-2 files to ensure OCR is working
- For heavily formatted PDFs, the tool extracts the largest/boldest text as title

### Problem: Special Characters in Filenames

**Cause**: PDF titles contain characters not allowed in filenames (like `\ / : * ? " < > |`)

**Solution**: Renomee automatically replaces these with safe alternatives:
- Colons (`:`) → underscores (`_`)
- Slashes (`/`) → hyphens (`-`)
- Question marks (`?`) → removed

### Problem: Duplicate Filenames

**Cause**: Multiple PDFs have identical titles

**Solution**: Renomee automatically appends a number: `Document_Title.pdf`, `Document_Title_2.pdf`, etc.
