## Step-by-Step Guide {#steps}

### Step 1: Install Renomee

If you haven't installed Renomee yet, [download Renomee AI](https://renomeeai.com/en/download).

Supports Windows 10+ and macOS 10.15+.

---

### Step 2: Select Folder

1. Launch Renomee
2. Click **"Select Folder"** or drag a folder into the window
3. Renomee will automatically scan the files

![Select Folder Screenshot](../assets/screenshots/select-folder.png)

---

### Step 3: Enter Renaming Rule

In the rule input box, describe your need in natural language:

```plaintext
Rename PDF files based on their content
```

Or be more specific:

```plaintext
Rename PDF files to: Title_Author_Year.pdf
```

!!! tip "Rule Writing Tips"
    - Simple descriptions work best—no regex needed
    - Supports both English and Chinese
    - Can reference file metadata (date, author, etc.)

---

### Step 4: Preview and Execute

1. Click **"Preview"** to see the renaming results
2. Check if they meet your expectations
3. If satisfied, click **"Execute"**

![Preview Interface Screenshot](../assets/screenshots/preview.png)

✅ **Done!** Smart renaming of hundreds of files completed in seconds.
