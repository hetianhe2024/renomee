# PDF 文件重命名

PDF 是最常见的文档格式之一。Renomee AI 提供强大的 PDF 内容识别能力，帮助你快速整理大量文档。

## 🎯 核心功能

- **标题提取**：自动识别文档标题
- **OCR 支持**：处理扫描版 PDF
- **元数据读取**：提取作者、日期等信息
- **大文件优化**：支持最大 50MB 的 PDF
